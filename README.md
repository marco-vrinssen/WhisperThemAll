# CleanButtons

Minimalist World of Warcraft addon that removes all button glow effects from action bars.

## Features

- Removes spell activation alerts and glow effects
- Hides overlay glows on action buttons
- Suppresses assisted combat rotation highlights
- Works with all action bar configurations

## Installation

1. Download the addon
2. Extract to `World of Warcraft/_retail_/Interface/AddOns/`
3. Restart WoW or reload UI with `/reload`

## Purpose

Eliminates distracting button glow animations that highlight abilities, providing cleaner action bars. Useful for players who prefer minimal visual feedback or already know their rotation.

## Compatibility

- **Interface Version**: 11.2.0 (The War Within)
- **Conflicts**: May conflict with action bar addons that modify button glow behavior
